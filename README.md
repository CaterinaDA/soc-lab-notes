# SOC Lab Notes

Practical security analysis exercises performed on a local Ubuntu Server lab.

## Environment
- Host: Windows 11
- VM: Ubuntu Server 26.04 LTS (VirtualBox)
- Connection: SSH

## Scenarios
- [01] SSH Brute Force Attack Detection
- [02] Linux Permissions and Processes
- [03] Networking: Ports and Connections
- [04] Suspicious Process Investigation
